Innovation Under Constraints

We talk a lot about "innovation under constraints," but let's talk about the practical side of it.

I run a 15-year-old computer and drive a 17-year-old car. My hobby budget is exactly $0. But instead of letting that limit me, it forced me to learn.

By leveraging free LLMs and open-source GitHub repositories, I’ve been diving deep into Python, Rust, C++, and JUCE. When a tool I need requires a commercial license, I treat it as a coding challenge and build my own custom solution.

You don't need top-tier hardware or expensive software to build powerful things. You just need curiosity (and maybe a little bit of stubbornness).

ExcelBridge — More Details

ExcelBridge is a high-performance, quant-ready extension stack for Excel that I’ve been building since 2023 — a lightweight local REST host that loads Python plugins at runtime and transforms Excel into a seamless front-end for modern numerical computing.

With years of experience building C++ XLLs over XLW, I designed ExcelBridge to avoid the bloat and fragility of traditional add-ins. It’s self-contained — no Python or Conda installation required for end-users.

ExcelBridge ships with NumPy, SciPy, Pandas, Polars, SymPy, scikit-learn, XGBoost, and more. It’s not just for Excel — its REST API means any HTTP client (curl, dashboards, services) can call your Python functions.

Workflow: Excel → GBPY VBA add-in → ExcelBridge → plugin → JSON → Excel spill. Clean, fast, and developer-friendly.
